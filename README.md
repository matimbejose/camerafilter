##  Camera filter
A camera filter app that updates emoji based on real-time face behavior.


## TECHNOLOGIES USED
- TypeScript
- React native (Expo)
- Expo camera
- Expo face detector
- React native reanimated


## How do runing (Expo users)
- clone the repository
- no diretório deste repositório execute estes comandos:
- a)yarn  install
- b) yarn starrt
